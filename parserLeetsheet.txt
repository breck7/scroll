Parsers LeetSheet
=================

*Parsers* is the advanced language Scroll developers use to extend the Scroll language with new parsers. Parsers files end in *.parsers*. Book available: Parsers Explained.
 https://parsers.breckyunits.com/ Parsers Explained

// - A comment.
baseParser - Set for blobs or errors. 
boolean - A constant.
catchAllCellType - Use for lists.
catchAllParser - Attach this to unmatched lines.
cells - Set required cellTypes.
crux - Attach by matching first word.
cruxFromId - Derive crux from parserId.
description - Parser description.
example - Set example for docs and tests.
extends - Extend another parser.
float - A constant.
inScope - Parsers in scope.
int - A constant.
javascript - Javascript code for Parser Actions.
listDelimiter - Split content by this delimiter.
pattern - Attach via regex.
popularity - Parser popularity.
required - Assert is present at least once.
root - Set root parser.
single - Assert used once.
string - A constant.
tags - Custom metadata.
uniqueFirstWord - Assert unique first words. For pattern parsers.
uniqueLine - Assert unique lines. For pattern parsers.

Parser Phases: Assemble -> Acquire -> Analyze -> Act
